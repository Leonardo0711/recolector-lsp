# Recolector de Lenguaje de Señas Peruano (LSP)

Este proyecto es una aplicación web diseñada para recolectar datos en video y coordenadas (usando MediaPipe) de Lenguaje de Señas Peruano, con el fin de generar un dataset estandarizado para entrenar modelos de Inteligencia Artificial.

## Características planeadas:
- Autenticación o registro de usuario simple.
- Interfaz para grabar palabras o frases (automáticamente convertidas a MAYÚSCULAS).
- Integración de cámara web (MediaPipe para tracking opcional).
- Compresión de grabaciones en formato de video (`.webm`).
- Integración para subida de archivos a Google Drive.
